# secure-authentication-system
“Built a Secure Authentication System using Flask with password hashing (bcrypt) and session management.”

A Flask-based authentication system with login and signup functionality.

## 🔧 Features
- User Signup & Login
- Password Hashing
- Session Management
- Logout system

## 🛠️ Tech Stack
- Python
- Flask
- SQLite

## ▶️ How to Run
1. Install dependencies:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
